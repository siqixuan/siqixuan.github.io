1.冒泡排序

我的答案：

# 方法一：
# 使用列表的sort方法
# 数据按照升序排序
nums=[3,7,8,1,2,9,5,6,4]
print(nums.sort())     # [1,2,3,4,5,6,7,8,9]
# 方法二：
# 使用while循环的冒泡排序法
i=0
while i<len(nums)-1:
    j=0
    while j<len(nums)-1-i:
        if nums[j]>nums[j+1]:
            nums[j],nums[j+1]=nums[j+1],nums[j]
        j+=1
    i+=1
print(nums)           # [1,2,3,4,5,6,7,8,9]

23.最大公约数(Largest Common Divisor,LCD)

最大公因数，也称最大公约数、最大公因子，指两个或多个整数共有约数中最大的一个。a，b的最大公约数记为（a，b），同样的，a，b，c的最大公约数记为（a，b，c），多个整数的最大公约数也有同样的记号。求最大公约数有多种方法，常见的有质因数分解法、短除法、辗转相除法、更相减损法。

几个整数中公有的约数，叫做这几个数的公约数；其中最大的一个，叫做这几个数的最大公约数。例如：12、16的公约数有1、2、4，其中最大的一个是4，4是12与16的最大公约数，一般记为（12，16）=4。12、15、18的最大公约数是3，记为（12，15，18）=3。

1.辗转相除法

其算法过程为： 前提：设两数为a,b设其中a 做被除数,b做除数，c为余数

  1. 大数放a中、小数放b中；
  2. 求a/b的余数c；
  3. 若c=0,则b为最大公约数；
  4. 如果c!=0则把b的值给a、c的值给a；
  5. 返回第二步；

2.相减法

其算法过程为：有两整数a和b：

  1. 若a>b，则a=a-b;
  2. 若a<b，则b=b-a;
  3. 若a=b，则a（或b）即为两数的最大公约数;
  4. 若a≠b，则再回去执行第一步；

我的答案：

# 方法一 利用辗转相除法求最大公约数
a=int(input("请输入一个整数："))
b=int(input("请输入另一个整数："))
c=a%b
while c!=0:
    a=b
    b=c
    c=a%b
print("%d和%d的最大公约数是%d" % b)
"""

最大公约数是3
"""

    if a>b:
        c=a-b

穷举法（也叫枚举法）穷举法求两个正整数的最大公约数的解题步骤：从两个数中较小数开始由大到小列举，直到找到公约数立即中断列举，得到的公约数便是最大公约数 。

其算法过程为：

  1. 求a,b中的较小数smaller；
  2. 若a,b能同时被smaller整除，则smaller即为最大公约数，结束；
  3. 若a,b不能同时被smaller整除，则进行下一轮循环，直到满足第二步条件，程序结束。

参考答案：

# 利用穷举法求最大公约数
def hcf(x,y):
    """
    该函数返回两个数的最大公约数
    """
    # 获取最小值
    if x>y:
        smaller=y
    else:
        smaller=x
    for i in range(1,smaller+1):
        if (x%i==0) and (y%i==0):
            hcf=i
    return hcf

a=int(input("请输入一个整数："))
b=int(input("请输入另一个整数："))
print(a,"和",b,"的最大公约数为",hcf(a,b))

"""

24.最小公倍数（Least Common Multiple，LCM）

两个或多个整数公有的倍数叫做它们的公倍数，其中除0以外最小的一个公倍数就叫做这几个整数的最小公倍数。整数a，b的最小公倍数记为[a，b]，同样的，a，b，c的最小公倍数记为[a，b，c]，多个整数的最小公倍数也有同样的记号。最小公倍数计算方法：分解质因数法和公式法。

公式法：两个自然数的乘积等于这两个自然数的最大公约数和最小公倍数的乘积。最小公倍数的计算要把三个数的公有质因数和独有质因数都要找全，最后除到两两互质为止。(a,b)x[a,b]=ab(a,b均为整数)

我的答案：

# 利用公式法，最小公倍数=两整数的乘积÷最大公约数
def hcf(x,y):
    """
    该函数返回两个数的最大公约数
    """
    # 获取最小值
    if x>y:
        smaller=y
    else:
        smaller=x
    for i in range(1,smaller+1):
        if (x%i==0) and (y%i==0):
            hcf=i
    return hcf

a=int(input("请输入一个整数："))
b=int(input("请输入另一个整数："))

# 穷举法求最小公倍数
def lcm(x,y):
    # 获取最大值
    if x>y:
        greater=x
    else:
        greater=y
    while True:
        if ((greater%x==0) and (greater%y==0)):
            lcm=greater
            break
        greater+=1
    return lcm

# 引入datetime模块
import datetime
def getYesterday():
    today=datetime.date.today()
    oneday=datetime.timedelta(days=1)
    yesterday=today-oneday
    return yesterday

print(getYesterday())      # 2020-10-03
